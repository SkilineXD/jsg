JavaScript Graphics

JavaScript Graphics (JSG) é uma API (Application Programming Interface) para o desenvolvimento de gráficos tridimensionais (3D) em JavaScript e HTML5 com WebGL.


JSG abstrai a complexidade de desenvolvimento de aplicações que usam gráficos 3D com WebGL por meio de abstrações bem estabelecidas.
